# kuune.org ![](https://github.com/naoina/kuune.org/workflows/Build%20and%20deploy/badge.svg)

My own profile, text (blog) site.

## Requirements

* [Go](http://golang.org/)
* [Hugo](http://gohugo.io/)

## Setup

```bash
go get -u github.com/gohugoio/hugo
```

## Create a text entry

Run `hugo new text/title-of-text.md`.
